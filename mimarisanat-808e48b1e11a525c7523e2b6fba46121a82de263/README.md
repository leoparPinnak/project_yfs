# Mimari Sanat - Berel Trio

Modern mimari ve tasarım projelerini sergilemek üzere geliştirilmiş, dinamik içerikli ve şık tasarımlı bir web uygulamasıdır. İçerisinde yer alan **Nokia Lumia Style Live Tiles** modülü ve gelişmiş **Yönetim Paneli** ile benzersiz bir kullanıcı deneyimi sunar.

## 🚀 Öne Çıkan Özellikler

- **Nokia Lumia Style Live Tiles:** Windows Phone estetiğini yansıtan, pürüzsüz kaydırma geçişleri ve Ken Burns tarzı yakınlaştırma (zoom) efektlerine sahip interaktif galeri.
- **Dinamik Yönetim Paneli:** Hero bölümünden projelere, özelliklerden iletişim bilgilerine kadar tüm sitenin içeriğini tek bir panelden yönetme imkanı.
- **Canlı Önizleme:** Yönetim panelinde yapılan değişikliklerin (örneğin zoom şiddeti veya döngü süresi) anlık olarak simüle edildiği canlı önizleme ekranı.
- **GPU Hızlandırmalı Animasyonlar:** Akıcı bir görsel deneyim için optimize edilmiş transformasyonlar.
- **Duyarlı (Responsive) Tasarım:** Tüm modern tarayıcılar ve mobil cihazlarla tam uyumlu.

## 🛠 Teknoloji Yığını

- **React & TypeScript**
- **Vite** (Hızlı geliştirme ve build süreci)
- **Lucide React** (Modern ikon kütüphanesi)
- **Vanilla CSS** (Tam kontrol ve yüksek performans için)
- **React Router** (Gelişmiş navigasyon)

## 💻 Yerel Geliştirme

Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Uygulamayı başlatın (development mode):
   ```bash
   npm run dev
   ```

3. Tarayıcınızda şu adresi açın: `http://localhost:3000`

## 📦 Proje Yapısı

- `/components`: Projenin yapı taşlarını oluşturan modüler React bileşenleri.
- `/pages`: Sayfa düzeyindeki bileşenler (Ana Sayfa, Admin Paneli, vb.).
- `/context`: Site içeriğini ve temasını yöneten merkezi state yapısı.
- `/public`: Statik varlıklar ve merkezi içerik dosyası (`site-content.json`).
